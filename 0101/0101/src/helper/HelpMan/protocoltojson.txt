/usr/bin/protocoltojson
Usage: protocoltojson [options] [files...]
Converts .protocol files to json

Options:
  -v, --version        Displays version information.
  -h, --help           Displays this help.
  -o, --output <name>  Output file name for JSON data.

Arguments:
  files                .protocol files to read.
