/usr/bin/systemd-stdio-bridge
systemd-stdio-bridge [OPTIONS...]

STDIO or socket-activatable proxy to a given DBus endpoint.

  -h --help              Show this help
     --version           Show package version
  -p --bus-path=PATH     Path to the kernel bus (default: unix:path=/run/dbus/system_bus_socket)
  -M --machine=MACHINE   Name of machine to connect to
