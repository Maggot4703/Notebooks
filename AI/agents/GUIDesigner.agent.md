# GUI Designer Agent

## Purpose
This agent is responsible for assisting with graphical user interface (GUI) design tasks, including layout suggestions, component selection, and best practices for user experience.

## Usage
- Use this agent for any workflow or automation that requires GUI design expertise.
- The agent can provide recommendations, generate mockups, and review existing designs.

## Example Tasks
- Suggesting layout improvements for a form.
- Recommending UI components for a dashboard.
- Reviewing accessibility of a user interface.

## Notes
- Formerly known as DESIGNER agent.
