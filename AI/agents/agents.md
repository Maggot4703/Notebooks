# Agent List

## Explore
Fast read-only codebase exploration and Q&A subagent. Safe to call in parallel. Specify thoroughness: quick, medium, or thorough.

## modernize-azure-dotnet
Modernize the .NET application.

## modernize-azure-java
Modernize the Java application.

## modernize-batch-impl
Execute code changes for a single batch of tasks, following the implement skill with full context awareness.

## modernize-design
Run the design phase — dynamically dispatch research sub-agents based on project type, then generate the specification document.

## modernize-foundation
Orchestrate foundation phase — constitution generation and knowledge graph building.

## modernize-gatekeep
Orchestrate a comprehensive cross-check for feature specifications, plans, and tasks.

## modernize-implementation
Orchestrate batch-based implementation by reading context, deciding batching strategy, dispatching each batch to BatchImplAgent, and managing state between batches.

## modernize-java
Upgrades Java projects to target versions (e.g., Java 21, Spring Boot 3.2) via incremental planning and execution.

## modernize-java-assessment
Assess codebases with evidence-based findings.

## modernize-java-upgrade
Upgrades Java projects to target versions (e.g., Java 21, Spring Boot 3.2) via incremental planning and execution.

## modernize-plan
Orchestrate implementation plan generation from specification with traceability checkpoint.

## modernize-rearchitecture
Orchestrate a workflow to re-architect an application module by coordinating specialized sub-agents.

## modernize-task
Generate tasks for a SINGLE phase/slice from the implementation plan. Designed to run in parallel — one instance per phase.


## GUIDesigner
Code-only agent for generating Python GUI code (Tkinter, PyQt, etc.) based on user prompts. User-driven, no visual tools—returns ready-to-run code snippets for GUI design. Formerly known as Designer.

## Traveller
Agent for exploring, analyzing, and working with Traveller RPG sector/world data, including UWP, sector files, and world attributes. Supports data modeling, search queries, and RPG-specific conventions.
