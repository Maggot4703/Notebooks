# BASH Workspace

This folder contains shell command documentation, exploration notebooks, and related scripts.

## Purpose
- Document and experiment with shell commands and workflows
- Provide reference notebooks for common shell tasks

## Usage
- Open `bash.ipynb` for interactive shell command examples
- Use the `commands/` subfolder for organized command documentation
- Run `uv sync && uv run jupyter lab` to start the environment

## Conventions
- Keep scripts and docs organized by topic in subfolders
- Prefer reproducible, copy-paste friendly command examples
- Use markdown for explanations and code cells for commands
