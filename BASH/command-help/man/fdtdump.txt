NAME
       fdtdump - prints a readable version of a flat device-tree file.

SYNOPSIS
       /usr/bin/fdtdump <DTB-file-name>

DESCRIPTION
       The  fdtdump  program  prints  a readable version of a flat device-tree
       file.
