NAME
       gpiodetect - manual page for gpiodetect v1.6.3

SYNOPSIS
       gpiodetect [OPTIONS]

DESCRIPTION
       List all GPIO chips, print their labels and number of GPIO lines.

OPTIONS
       -h, --help:
              display this message and exit

       -v, --version:
              display the version and exit
