NAME
       gpiofind - manual page for gpiofind v1.6.3

SYNOPSIS
       gpiofind [OPTIONS] <name>

DESCRIPTION
       Find a GPIO line by name. The output of this command can be used as in‐
       put for gpioget/set.

OPTIONS
       -h, --help:
              display this message and exit

       -v, --version:
              display the version and exit
