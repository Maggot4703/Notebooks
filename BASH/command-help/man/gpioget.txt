NAME
       gpioget - manual page for gpioget v1.6.3

SYNOPSIS
       gpioget [OPTIONS] <chip name/number> <offset 1> <offset 2> ...

DESCRIPTION
       Read line value(s) from a GPIO chip

OPTIONS
       -h, --help:
              display this message and exit

       -v, --version:
              display the version and exit

       -l, --active-low:
              set the line active state to low

       -B, --bias=[as-is|disable|pull-down|pull-up] (defaults to 'as-is'):

              set the line bias

   Biases:
       as-is: leave bias unchanged

       disable:
              disable bias

       pull-up:
              enable pull-up

       pull-down:
              enable pull-down
