NAME
       gpioinfo - manual page for gpioinfo v1.6.3

SYNOPSIS
       gpioinfo [OPTIONS] <gpiochip1> ...

DESCRIPTION
       Print information about all lines of the specified GPIO chip(s) (or all
       gpiochips if none are specified).

OPTIONS
       -h, --help:
              display this message and exit

       -v, --version:
              display the version and exit
