NAME
       spa-inspect - spa-inspect
        The PipeWire SPA plugin information utility

SYNOPSIS
       spa-inspect FILE

DESCRIPTION
       Displays information about a SPA plugin.

       Lists the SPA factories contained, and tries to instantiate them.

EXAMPLES
       spa-inspect /usr/lib/aarch64-linux-gnu/spa-0.2/bluez5/libspa-codec-
       bluez5-sbc.so

       Display information about a plugin.
