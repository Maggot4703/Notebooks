NAME
       spa-json-dump - spa-json-dump
        SPA JSON to JSON converter

SYNOPSIS
       spa-json [FILE]

DESCRIPTION
       Reads a SPA JSON file or stdin, and outputs it as standard JSON.

EXAMPLES
       spa-json-dump /usr/share/pipewire/pipewire.conf
