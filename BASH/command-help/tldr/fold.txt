fold

[0m[0mWrap each line in an input file to fit a specified width and print it to stdout.
More information: https://manned.org/fold.1p.

 - [0m[32;23;22;24;25mWrap each line to default width (80 characters):
   [33;23;22;24;25mfold [0mpath/to/file
[0m
 - [0m[32;23;22;24;25mWrap each line to width "30":
   [33;23;22;24;25mfold -w30 [0mpath/to/file
[0m
 - [0m[32;23;22;24;25mWrap each line to width "5" and break the line at spaces (puts each space separated word in a new line, words with length > 5 are wrapped):
   [33;23;22;24;25mfold -w5 -s [0mpath/to/file
