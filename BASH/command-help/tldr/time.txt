time

[0m[0mMeasure how long a command took to run.
Note: time can either exist as a shell builtin, a standalone program or both.
See also: times.
More information: https://manned.org/time.

 - [0m[32;23;22;24;25mRun the command and print the time measurements to stdout:
   [33;23;22;24;25mtime [0mcommand
[0m
 - [0m[32;23;22;24;25mCreate a very simple stopwatch (only works in Bash):
