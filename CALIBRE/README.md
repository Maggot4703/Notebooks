# CALIBRE Workspace

This folder contains notebooks and scripts for working with Calibre (e-book management) tools and workflows.

## Purpose
- Document Calibre CLI usage and automation
- Explore e-book conversion, metadata, and library management

## Usage
- Open `calibre.ipynb` for interactive Calibre command examples
- Reference `help-calibre.txt` and `man-calibre.txt` for command help
- Run `uv sync && uv run jupyter lab` to start the environment

## Conventions
- Store Calibre-related scripts and docs in this workspace
- Use markdown for documentation and code cells for automation
