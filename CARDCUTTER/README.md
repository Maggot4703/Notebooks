# CARDCUTTER Workspace

This folder contains tools, notebooks, and assets for card-cutter image processing and related workflows.

## Purpose
- Batch process images for card and grid overlays
- Manage and document card-cutter scripts and data

## Usage
- Open `card-cutter.ipynb` for interactive examples
- Use the `CardCutter/` subfolder for main scripts and data files
- Run `uv sync && uv run jupyter lab` to start the environment

## Conventions
- Organize scripts and data by function (e.g., image processing, CSVs)
- Document workflows in markdown and notebooks
