# JUPYTERLAB Workspace

This folder is for exploring JupyterLab extensions, configuration, and advanced notebook workflows.

## Purpose
- Test and document JupyterLab features and extensions
- Experiment with notebook UI and configuration

## Usage
- Open `jupyterlab.ipynb` for interactive JupyterLab examples
- Store extension tests and configs in `jupterlab_files/`
- Run `uv sync && uv run jupyter lab` to start the environment

## Conventions
- Keep extension and config experiments organized by topic
- Use markdown for documentation and code cells for demos
