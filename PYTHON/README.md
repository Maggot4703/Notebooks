# PYTHON Workspace

This folder is for general Python learning, experimentation, and notebook-based workflows.

## Purpose
- Practice Python coding and data analysis
- Document Python language features and libraries

## Usage
- Open `python.ipynb` for interactive Python examples
- Store scripts and notes in this workspace
- Run `uv sync && uv run jupyter lab` to start the environment

## Conventions
- Organize notebooks and scripts by topic or library
- Use markdown for explanations and code cells for code
