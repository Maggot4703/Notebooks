# TRAVELLERMAP Data Directory

This folder stores sector/world data files for Traveller map tools and scripts.

## Purpose
- Hold all data files used by TravellerMap scripts (e.g., sector, world, UWP data)

## Usage
- Place sector/world data files here for use by scripts in the parent directory
- Reference this directory in scripts and notebooks for data loading

## Conventions
- Use clear, descriptive filenames for data files
- Keep raw and processed data organized in subfolders if needed
