# Accessibility: Screen Reader & Keyboard Usage

## How to Use Traveller Tools with Screen Readers

- All CLI tools support plain text and CSV output for compatibility with screen readers.
- Use the `export_plaintext_csv.py` script to convert sector/world data to accessible formats.
- For best results, use a monospace font and enable line wrapping in your terminal or text editor.
- All prompts and outputs are designed to be unambiguous and screen-reader friendly.

## Keyboard Navigation

- All interactive scripts are fully operable via keyboard (no mouse required).
- Use Tab/Shift+Tab to move between fields (if applicable).
- Press Enter to activate or confirm actions.
- No element traps keyboard focus; you can always Tab out.

## Additional Tips

- For web/HTML outputs, use browser accessibility features (reader mode, zoom, high-contrast mode).
- Refer to the accessibility checklist for more guidance.
