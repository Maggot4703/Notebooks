# TRAVELLERMAP Output Directory

This folder is for generated or exported files from TravellerMap scripts and notebooks.

## Purpose
- Store output files such as reports, images, or processed data

## Usage
- Scripts and notebooks should write outputs here
- Clean up or archive old outputs as needed

## Conventions
- Use descriptive filenames and subfolders for outputs
- Keep outputs separate from raw data and scripts
