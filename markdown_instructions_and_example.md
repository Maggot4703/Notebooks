# Markdown Instructions and Examples

## What is Markdown?
Markdown is a lightweight markup language for formatting text. It is widely used for README files, documentation, and note-taking because it is easy to read and write.

## Basic Syntax

- **Headings:**
  - `# Heading 1`
  - `## Heading 2`
  - `### Heading 3`
- **Bold:** `**bold text**` → **bold text**
- **Italic:** `*italic text*` → *italic text*
- **Lists:**
  - Unordered: `- Item 1`  `* Item 2`
  - Ordered: `1. First`  `2. Second`
- **Links:** `[link text](https://example.com)`
- **Images:** `![alt text](image.png)`
- **Code:**
  - Inline: `` `code` ``
  - Block:
    ```
    code block
    ```
- **Blockquote:** `> quoted text`
- **Horizontal Rule:** `---`

## Example

```
# My Project

This is a **Markdown** example.

## Features
- Easy to use
- Fast to write

## Usage
1. Clone the repo
2. Edit the files

> Markdown is great for documentation!
```

## Tips
- Preview Markdown in VS Code: Right-click the file and select "Open Preview" or press `Ctrl+Shift+V`.
- Export to PDF: Use the Markdown Preview Enhanced extension or Pandoc for best results.
